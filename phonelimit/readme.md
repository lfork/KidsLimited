# Normal(Pro) Version 

## Some Device conditions
+ rooted
+ uninstalled the root applications, like Root Explore

## Working Principles

#### 1.程序需要一直运行(前台进程)
#### 2.Pro版本获取root权限
#### 3.需要查看程序使用情况的权限
#### 4.开启限制的时候需要默认桌面或者悬浮窗的权限

## Some Bugs
+ 1、在息屏的情况下手机可能会被优化掉，这个时候启动定时服务的话就会报错
+ 2、在息屏的状态下计时可能会不准确，会提前结束计时。
